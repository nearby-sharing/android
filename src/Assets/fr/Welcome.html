Bienvenue

Cette application vous permet d’envoyer et de recevoir des fichiers ou des liens avec des appareils Windows 10 / 11

Configuration - Android

  • Veuillez activer le Bluetooth!
  • L'accès a la localisation est requise pour rechercher des appareils à proximité.

Configuration - Windows

  • Veuillez suivre les instructions sur https://nearshare.shortdev.de/docs/setup sur votre appareil Windows

Envoyer

  • Vous pouvez envoyer des données depuis n'importe quelle application avec le menu de partage ou avec le bouton ci-dessous.
  • Vous devriez voir une notification sur votre appareil Windows pour accepter le partage.

Recevoir

  • Vous pouvez recevoir les fichiers en cliquant sur le bouton ci-dessous ou en utilisant le bouton d'action.
