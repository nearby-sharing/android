Willkommen!

Mit dieser App können Sie Dateien und Websites an und von "Windows 10/11" Geräten senden und empfangen.

Einrichtung – Android

  • Bitte aktivieren Sie Bluetooth!
  • Die Standortberechtigung ist erforderlich, um nach Geräten in der Nähe zu suchen.

Einrichtung - Windows

  • Bitte folgen Sie den Anweisungen unter https://nearshare.shortdev.de/docs/setup auf Ihrem Windows-Gerät.

Senden

  • Sie können Daten senden, indem Sie das native Android-Freigabemenü oder die Schaltfläche unten verwenden.
  • Sie sehen eine Benachrichtigung, die von Ihrem Windows-Betriebssystem selbst gesendet wird, um die Übertragung zu akzeptieren.

Empfangen

  • Sie können Dateien erhalten, indem Sie auf die Schaltfläche unten klicken oder die Schnellkachel verwenden.
