¡Bienvenido!

Esta aplicación te permite enviar y recibir archivos y páginas web hacia y desde dispositivos Windows 10 / 11.

Configuración inicial - Android

  • ¡Por favor, activa el Bluetooth!
  • El permiso de ubicación es necesario para buscar dispositivos cercanos.

Configuración inicial - Windows

  • Por favor, sigue las instrucciones en https://nearshare.shortdev.de/docs/setup en tu dispositivo Windows.

Enviando

  • Puedes enviar datos utilizando el menú nativo de compartir de Android o el botón de abajo.
  • Verás una notificación enviada desde el propio Windows para que aceptes la transferencia.

Recibiendo

  • Puedes recibir archivos pulsando sobre el botón de abajo o utilizando el botón en el panel de ajustes rápidos del dispositivo.
