Diese Funktion befindet sich derzeit in der Alpha-Phase. Bitte erwarten Sie langsame und unzuverlässige Übertragungen sowie andere Arten von Fehlern.
Sie können derzeit nur eine (kleine) Datei gleichzeitig empfangen, und der einzige verfügbare Transport ist Bluetooth.
Wenn Sie keine Daten empfangen können, überprüfen Sie die Einstellungen, das Debug-Protokoll und werfen Sie einen Blick in die FAQ (in den Einstellungen).
Wenn Sie auf ein Problem stoßen, melden Sie es bitte direkt über GitHub!
https://github.com/ShortDevelopment/Nearby-Sharing-Windows/issues

Erforderliche Aktion


Aufgrund von Datenschutzbedenken (Werbung) erlaubt Android Apps nicht mehr, Ihre Bluetooth-Mac-Adresse zu lesen.
Es ist jedoch erforderlich, dem anderen Gerät mitzuteilen, wohin es die Daten senden soll (wie eine Haus-Adresse).
Um fortzufahren, geben Sie bitte unten Ihre Bluetooth-Mac-Adresse ein.
Sie finden sie in den Einstellungen unter System > Über >Bluetooth-MAC-Adresse.
Ist diese Adresse ungültig, können Sie keine Daten empfangen!
